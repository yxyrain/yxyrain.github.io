实例化和具体化

了解实例化和具体化的前提是熟悉模板。

  • 隐式实例化

    • 通过名字可见，就是清楚的表明你要实例化的类型
  • 显式具体化
    • 因为对于某些特殊类型，可能不适合模板实现，需要重新定义实现，此时就是使用显示具体化的场景

编译防护方案
  • 使用 #ifndef 防止重复编译

    此种方式代表如果从未定义过 QAQ_H_ 这个变量，才执行 #ifndef QAQ_H_ 到 #endif 之间的语句，由于第一次编译时会 #define QAQ_H_ 所以多次编译时，此模块只会执行一次，防止了重复编译。

存储持续性 作用域和链接性

存储类别如何影响信息在文件间的共享

  • 自动存储持续性
    • 代表：局部变量，无链接性
    • 存储地点：留出一段内存，构造一个栈，使用结束后，将其释放
    • 寿命短，代码块结束便被释放
    • register ： 以前是建议 CPU 寄存器存储变量，现在加不加一样
  • 静态存储持续性
    • 代表：全局变量 z ， static 变量 q ， 区间 static 变量 k
    • 特点：默认零初始化， 寿命长，一直持续到进程终结
    • 储存地点：固定的内存块，不使用栈进行操作
    • 静态的持续性，外部的链接性：
      • C++ 的 单定义规则(ODR)：变量只能有一次定义机会
      • 两种定义声明
        1. 定义声明：int a;
        2. 引用声明：extern int a;
      • 使用方法：多文件使用外部变量时，用关键字 extern 声明
    • C++的作用域解析运算符 $::$ ， 放在变量名前代表全局版本的变量
    • static 限定符:当用于作用域为全文件的变量时，会使该变量的链接性变为内部链接性
      • 用法：在两个文件中创建名称相同的变量显然是不合乎标准的，这时，使用 static 会使另一个文件的常规外部变量隐藏。

说明符和限定符

  • cv-限定符
    1. const
      • 表明内存被初始化后，程序无法对其进行更改
      • 在 C++ 中， const 全局变量的链接性是内部的
      • const 的内部链接性可以用 extern 进行覆盖
    2. volatile
      • 表明即使程序代码没有对内存进行修改，值依然有可能变化，例如指针指向硬件位置，其中包含了来自串行端口的时间信息，可以避免程序将这个值缓存到寄存器中，从而出现两次同样的值
    3. mutable
      • 可以用 mutable 指出，即使结构体/类变量为 const ，其中的某个成员依然可以被修改

        函数和链接性

  • 所有函数的储存链接性都自动设为静态，且链接性默认为外部链接
  • 可以使用 extern 来说明函数是另外一个文件定义的
  • 可以使用 static 将函数的链接性设置成内部
  • 内联函数不受单定义规则的约束

储存方案和动态分配

  • 使用 new / malloc() 函数分配的内存被称为动态内存
  • 编译器使用三块独立的内存
    • 静态变量（可能再细分）
    • 自动变量
    • 动态存储
  • new 的使用
    • 使用 new 运算符初始化
    • new 失败时，返回异常 std::bad_alloc
    • new 运算符，函数，替换函数
    • 定位 new 运算符
      • 定位 new 作用为指定一个位置获取内存，但要注意，用定位 new 获取的内存无法被 delete 释放，delete 只能释放 heap（堆） 中获取的内存 上述代码代表在 buffer1 的地址上创建 p1 ， 在buffer2 的地址上 创建 p2 数组

一些感受

发表于 2020-12-02 分类于 当我发呆时我在想什么

我人生的第二十个十一月

其实我有想过2020是比较痛苦的一年，但是我实在没想到，原来这么痛苦。

前天是11月的最后一天，这个11月也许能够成为我前20年人生中排名前列的、让我感受深刻的11月。

病痛

每当我的肛周出现大大小小的毛病，都会让我不由得回想起2018年那个痛苦的春天。而每当我自认为我可以轻快的跳出当年那个让我痛苦的、不想回想起的记忆的瞬间，它又会悄悄的跟回来。偶尔刺痛一下我，提醒我不要忘了：其实它一直都还在。

我实在没法子了，内心无比纠结——谁想进医院呢？

进医院就意味着高昂的检查费、严重的心理压力、以及那种妄图把人吞掉的氛围。

但是没有办法，我意志力再如何坚强，也摆脱不了—-我终究只是一个普通的、有着私心的、无法忍受病痛折磨的人罢了。

决定了要去医院看病，首先要解决的就是钱了，我平时虽然花钱大手大脚，但也从来没有乱花过—–毕竟生活费有限，现在要我拿出小几千来看病，又谈何容易呢。

于是，我几乎是窒息着下定了决心向父母要钱，我知道如果我要，他们一定会给，但是想想家里微薄的收入，我几乎是强忍着泪水打下的一串汉字“妈妈给我钱我想看看我的肠胃”。

我太了解我的父母了，他们一定会慌不择路然后抓紧给我打电话问我怎么了，然后再一股脑的东拼西凑出足够支撑我看病的费用。正因为我是那么了解他们，我的眼泪再也忍不住了。我强忍着呜咽的声音和他们打完了电话。

唔，去看病吧。

打开了手机app预约挂号，找到了我想预约的科室，结果我直接傻眼了——-只是挂号就要100块，那一刻我像个从没见过世面的小孩子，呆住了。我生活了20年，上至三甲下至二乙，去过的医院不计其数。但是挂号要100块的，第一次见。我一边操作着手机，一遍不争气的流泪，我实在是不知道是什么样的医生，仅仅是挂一次号，见一次面，就要100块。那个时候我的想法其实有点可笑，我在想，我以后一定要挂100块的号心里没有波动。

挂好了号，如约检查，医生检查了10分钟，说没太大毛病，建议我做一个肠镜，我说好，于是交了核酸检测云云的费用，拿到了泻药，回到寝室为肠镜做准备。

当晚，我很痛苦的喝完了泻药，拉到了凌晨3点，但是我根本睡不着，因为病还在，我躺在床上胡思乱想，开始使用百度查我的病，我当然知道百度纯粹是胡诌八扯，但是人在那样的情况下，又谈何理智呢？只想在黑暗中找到一根稻草，然后使劲抓住，我也一样，我在睡前一直祈祷，看着百度上形形色色的病症，我的心跳一直加速，但是我明白，真的假不了，假的真不了。

第二天我去消化内镜中心等待着神的审判，我几乎拖着自己疲惫的身躯进了肠镜检查中心，肠镜的痛苦我不想赘述，因为身体上的痛苦远不如精神上的痛苦来的更加痛苦。我抱着胀气的肠子走出中心，等着审判，过了一会，拖着身子拿到了检查单———很健康。

啊。

心里的石头哐当一下，掉到了地上摔了个稀碎——我知道，神这次没惩罚我。

于是突然就开始想，今晚吃点什么庆祝一下。

然后我呆了一小会，排了排气，正准备叫上齐宝回寝室，我看到一个30出头的女性，抱着一个年龄相当的男性————哭了。不是那种痛哭，是那种很容易就能看出来她在克制、在勉强、在发抖的那种小心翼翼的哭。

我想，也许神惩罚她了。

我无意怜悯别人，因为我自己过的也并不顺心。但是看到那样的神情。我的内心还是不由得的为她祈祷——快点好起来吧，虽然我知道没用，但是大大小小的生命在我身边枯萎，随着年龄的增长，即使作为一个无神论者，我越来越懂得神存在的重要性。神可以是万病可治的药，可以是医术精湛的主刀医生，可以是一块牌匾，它甚至可以没有实体，但是它不能不存在。因为它是无数穷人的救命稻草，是全体绝症病人的精神期盼，总要有个盼头，你说对吧。

最后我虽然不忍心看着她哭，但是还是加快脚步走了出去，在医院门口，看到了很多在可移动手术病床上被盖住了面容的人，我想，他们都会变成星星，照亮地球乃至任何一个有生物存在的星球。

我的十一月就这么结束了。
